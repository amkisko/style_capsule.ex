# StyleCapsule Examples

This directory contains example applications demonstrating StyleCapsule usage.

## Example Applications

### Phoenix Demo App

A simple Phoenix application demonstrating:
- Basic component usage with StyleCapsule
- LiveView integration
- Different caching strategies
- Namespace usage

To run it, see `examples/phoenix_demo/README.md`.

### Standalone Elixir App

A framework-agnostic example showing:
- Direct CSS processing
- HTML wrapping
- Standalone usage patterns

To run it from the project root:

```bash
mix deps.get
mix run examples/standalone/example.exs
```

## Running Examples

Each example includes its own README with setup and running instructions.
